Run once — all current and future certs will be trusted automatically.

# Mac + iPhone — also opens AirDrop
bash scripts/trust-local-ca.sh --iphone

Why This Approach vs Alternatives
✓ cert-managerAuto-renewal, K8s-native, zero ops overhead. This is what we use. ~ Vault PKIBetter for multi-team, audit logging, short-lived certs. Overkill for homelab. ~ mkcertGood for dev, but manual renewal. We use it only to trust our CA on Mac. ✗ Self-signed (old)What we had before — manual, browser warnings, one secret per service.

Setup Roadmap

1. Migrate NUC backup data from sda to K8s PVCs via migration bridges
2. Install Talos v1.13.2 on sdc (Samsung 870 EVO)
3. Join GPU node as worker — enable NVIDIA device plugin + CUDA
4. Install Windows 11 on sda (ADATA SU650) for gaming
5. Configure dual-boot: BIOS selects Talos (sdc) by default, F8 for Windows
